The following must be done to use the GUI:

1) Install Python37
    Found at: https://www.python.org/downloads/
    Change PATH variable location of python.exe
2) Install PySerial
    Open Command Line
    'pip install pyserial'
3) Install PyQt5
    Open Command Line
    'pip install PyQt5'
4) Run the GUI
    Open Command Line
    'python CameraGUI.py'
    or
    'python CameraGUI.py -p <COM Port>'
